## 📌 프로젝트 소개

KB-Year는 여러 페이지를 이동하지 않고  
**재정 상태와 소비 흐름을 한눈에 파악할 수 있도록 만든 가계부 서비스**입니다.

---

## ✨ 주요 기능

- 📊 **재정 요약 카드**
  - 수입 / 지출 / 순수익을 한눈에 확인

- 📅 **캘린더 기반 소비 패턴**
  - 날짜별 수입/지출 확인
  - 특정 날짜 클릭 시 상세 내역 조회

- 🧾 **거래 내역 관리**
  - 거래 등록 / 수정 / 삭제

- 📈 **수입·지출 그래프**
  - 재정 흐름을 직관적으로 확인

- 🏅 **재정 상태 뱃지**
  - 소비 습관을 시각적으로 표현
